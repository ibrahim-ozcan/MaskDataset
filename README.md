# MaskDataset


This dataset is collected from public street cameras. It is composed of three class of objects. Each class has approximately 1.5k of face images. This dataset was used to train a face mask detection model. Dataset also fix the bearded man and the different kind of colored mask problem.


**1.REGULARLY WORN MASK**

Faces with regularly worn mask. It is accepted regularly when the mask covers the most of the nose and mouth.
